# Plantillas Aprobadas

Plantillas oficiales para comunicaciones y documentos: mensajes CRM (cobranza, seguimiento, recordatorio de compromiso), comunicaciones RH a candidatos, reportes operativos y financieros.

Regla crítica: **CRM_AGENT y RH_AGENT solo pueden redactar borradores basados en plantillas de este directorio en estado `approved`.** Contactar clientes o candidatos requiere además consentimiento y revisión humana.
