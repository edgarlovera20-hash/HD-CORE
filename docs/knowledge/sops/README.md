# SOPs — Procedimientos Operativos Estándar

Procedimientos paso a paso para operaciones recurrentes del ecosistema Heavenly Dreams.

Cada SOP debe incluir los metadatos definidos en `docs/KNOWLEDGE_HUB_STRATEGY.md`: documentId, version, status, ownerPlatform, approvedBy, aiUsageAllowed y sensitivity.

Los agentes IA solo consumen SOPs en estado `approved`.
