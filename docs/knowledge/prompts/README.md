# Prompts Aprobados

Prompts oficiales y versionados para los agentes IA del ecosistema (RH_AGENT, CRM_AGENT, SUPERVISOR_AGENT, FINANCE_AGENT, BRAIN_AGENT).

Reglas:

1. Ningún agente opera con prompts fuera de este directorio una vez en producción.
2. Cambios de prompt requieren PR y aprobación del owner de la plataforma.
3. Cada prompt declara: agente objetivo, fase IA permitida (lectura/borrador/solicitud), y restricciones de dominio.
