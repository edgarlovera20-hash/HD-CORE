# HD-CRM Domain

## Diagnóstico

HD-CRM controla clientes nuevos, clientes activos, seguimiento, cuentas vencidas, compromisos de pago, conversaciones, reportes y auditoría.

## Riesgos

- Contactos sin consentimiento o política aprobada.
- Seguimiento sin historial.
- Promesas de pago sin trazabilidad.
- Mezcla con Admin u Operations.

## Solución Propuesta

Separar CRM como dominio propio con contratos, permisos, eventos y auditoría.

## Módulos

- Clientes
- Seguimientos
- Conversaciones
- Compromisos de pago
- Reportes
- Auditoría

## Prioridad

Alta.
